This is an analytics project built with Next.js

## GELND

This app can help christians to measure, document and study their spiritual growth by setting session for their activities

![dashboard](./public/images/opengraph.png)

## How to Use

- Begin a timer when you start your Bible study or prayer time. Focus on God without distraction.
- Capture key revelations, prayers answered, and lessons learned during your time with God.
- Review your analytics to see your spiritual growth over time and celebrate your progress.

## Run Locally

- initialize a firbasse project
- install mongodb
- start development server

```bash
npm run dev
```
